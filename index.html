Objective 4.1: Basic Network Security Concepts

Question 1

Which of the following technologies are X.509 certificates most commonly associated with?

A. PKI

B. VLAN tagging

C. LDAP

D. MFA

Show Answer & Explanation ▶

Correct Answer: A. PKI

Why this is correct: X.509 certificates are the standard format for public key certificates used in PKI (Public Key Infrastructure) systems.

Why others are incorrect:

  • B: VLAN tagging is unrelated to certificates
  • C: LDAP is a directory service, not directly related to X.509
  • D: MFA is an authentication method, not certificate-based

Question 2

A. IAM

B. MFA

C. RADIUS

D. SAML

Show Answer & Explanation ▶

Correct Answer: D. SAML

Why this is correct: SAML (Security Assertion Markup Language) is an XML-based standard for exchanging authentication and authorization data between parties.

Why others are incorrect:

  • A: IAM is identity management but not XML-based
  • B: MFA is authentication method, not a data exchange protocol
  • C: RADIUS is authentication protocol but not XML-based

Why others are incorrect:

  • D: ARP poisoning manipulates MAC-IP mappings

Correct Answer: A. DNS poisoning

Why this is correct: DNS poisoning corrupts DNS records to redirect users to malicious sites.

Why others are incorrect:

  • B: DoS makes sites unavailable but doesn't redirect
  • C: Social engineering tricks users but doesn't alter DNS
  • D: ARP spoofing affects local network traffic

  • D: DNS poisoning affects domain resolution

Question 32
